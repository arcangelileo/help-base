Embeddable Search Widget

Add a search widget to your website so customers can find help articles without leaving your product.

Installation

Copy this code and paste it before the closing </body> tag on your website.

<!-- HelpBase Search Widget -->
<script src="{{ base_url }}/widget/{{ hc.slug }}/embed.js" async></script>

How It Works

1

Add the Script

Paste the embed code on any page where you want the help widget to appear.

2

Widget Appears

A floating help button appears in the bottom-right corner of your page, matching your brand color.

3

Instant Search

Customers search your help articles in real-time without leaving your website.

Widget Features

Brand color matching
Real-time search with debouncing
Keyboard shortcuts (Escape to close)
Mobile responsive
Highlighted search snippets
Lightweight — no dependencies
